## Getting Started

### Prerequisites
- Node.js 14+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/amra2026/amraSEOportfolio.git

# Navigate to project directory
cd amraSEOportfolio

# Install dependencies
npm install

# Start development server
npm start
```

### Build for Production

```bash
npm run build
```
